# 한계와 향후 계획

## 현재 한계

- 위치 공유가 인메모리 스토어 기반이라 서버리스 재시작 시 세션이 초기화될 수 있다.
- 전국 초정밀 버스 데이터는 도착예정시간(ETA)을 제공하지 않는다.
- 공공 API 키 미설정/응답 지연 시 일부 카드가 `demo/no-key/fallback` 소스로 동작한다.
- 경로 이탈 감지는 현재 규칙 기반(출발점 대비 비정상 이동)이며 정밀 경로 매칭은 미적용 상태다.

## 향후 계획

- Redis/Vercel KV 기반 세션 영속화
- 경로 엔진 연동 기반 이탈 감지 정확도 개선
- 푸시/문자 등 부모 경보 채널 확장
