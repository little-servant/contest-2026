# HonGa 데모 리허설 가이드

## 1) 준비물

- 스마트폰 2대 (권장) 또는 PC 1대 + 스마트폰 1대
- 인터넷 연결
- 배포 URL: `https://carepass-eight.vercel.app`

## 2) 리허설 목표 (총 6~10분)

- 아이 화면에서 목적지 선택 후 세션코드 생성 확인
- 부모 화면에서 코드 연결 후 실시간 위치 갱신 확인
- 도착 배너 표시 확인

## 3) 진입 URL

- 홈: `https://carepass-eight.vercel.app/`
- 아이 화면: `https://carepass-eight.vercel.app/child`
- 부모 화면: `https://carepass-eight.vercel.app/parent`

## 4) Step-by-Step

| 단계 | 조작 | 확인 포인트 | 예상 시간 |
|---|---|---|---|
| 0 | 기기 A(아이), 기기 B(부모)에서 각 URL 접속 | 두 화면 모두 정상 렌더링 | 30초 |
| 1 | 기기 A 홈에서 `아이로 시작` 클릭 | `/child`로 이동 | 20초 |
| 2 | 기기 A에서 `내 위치 찾기`(또는 `🎮 데모 시작`) 클릭 | 위치 확보 후 목적지 선택 화면 표시 | 30~60초 |
| 3 | 기기 A에서 `도서관` 선택 | Polyline, 버스 정보, 도서관 빈자리 카드 표시 | 60초 |
| 4 | 기기 A 상단 4자리 세션코드 확인 | 헤더 우측 `부모님 코드` 노출 | 10초 |
| 5 | 기기 B `/parent`에서 코드 입력 후 `연결하기` | 연결 배지 + 좌표 + 최근 갱신 시간 표시 | 40초 |
| 6 | 10~20초 대기 | 부모 화면이 5초 폴링으로 갱신되는지 확인 | 20초 |
| 7 | 목적지 도착 확인 | 부모 화면에 `목적지에 도착했어요!` 배너 표시 | 2~8분 |

## 5) 세션코드 생성 방법

- 세션코드는 아이 화면(`/child`) 진입 시 자동으로 생성된다.
- 코드는 목적지 선택 후 이동 화면 상단의 `부모님 코드` 영역에 표시된다.
- 형식은 4자리 숫자(`0000`~`9999`)다.

## 6) 심사위원 확인 포인트

- 아이 화면: `지도 Polyline + 버스 실시간 + 도서관 빈자리`가 동시에 표시되는가
- 부모 화면: `5초 폴링`, `연결 상태`, `최근 위치 좌표`가 갱신되는가
- 도착 이벤트: `목적지에 도착했어요!` 배너가 표시되는가

## 7) 시간 단축 팁

- `🎮 데모 시작` 버튼이 보이면 데모 모드로 빠르게 동선 설명이 가능하다.
- 발표 시간이 촉박할 경우 부모 화면 데모 패널의 `도착` 버튼으로 최종 배너를 먼저 확인하고, 자동 도착 흐름은 구두로 보완 설명한다.
