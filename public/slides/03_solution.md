# 솔루션

## HonGa 핵심 흐름

1. 아이가 `/child`에서 현재 위치를 확인하고 목적지(집/도서관)를 선택한다.
2. 카카오맵 Polyline, 버스 정보, 도서관 열람실 현황, 신호등 경보를 한 화면에서 확인한다.
3. 아이 화면의 4자리 세션코드를 부모가 `/parent`에서 입력한다.
4. 부모 화면이 5초 폴링으로 위치와 상태(정상/정지/이탈/연결끊김/도착)를 모니터링한다.

## 핵심 기능

- 아이 화면: GPS 진입, 목적지 선택, 실시간 데이터 기반 이동 안내, AI 음성 안내.
- 부모 화면: 실시간 위치 확인 + 상황 경보 배너.
- 위치 공유 API: `/api/location/update`, `/api/location/poll` 인메모리 세션 구조.
