# 데모 흐름

## 화면 흐름 4단계

1. 홈
   - 현재 위치 사용 또는 수동 좌표 입력
   - 근처 기관 진입
   - ![홈 화면](../../screenshots/home.png)

2. 기관 목록 / 지도
   - 기관 목록과 지도 연동
   - 마커 클릭 시 상세 페이지 이동
   - ![기관 목록/지도](../../screenshots/facilities.png)

3. 기관 상세
   - 기관 정보 확인
   - 이동 수단 요약 카드 확인
   - ![기관 상세 · 이동수단 요약](../../screenshots/route-summary.png)

4. 실시간 이동 정보
   - 교통약자 차량 현황
   - 전국 초정밀 버스 위치
   - ![실시간 이동 정보](../../screenshots/route-live.png)
